# 关于互联网与物联网

## 互联网与物联网区别

|      | 简介                                       | 意义           | 关键技术                 |
| ---- | ---------------------------------------- | ------------ | -------------------- |
| 互联网  | 解决信息共享、交互。连接了人和信息内容，提供标准化服务。             | 把卖产品变为卖内容和服务 | 大数据、云计算、商业智能         |
| 物联网  | 通过通信技术（射频识别 RFID），把传感器（红外感应、全球定位、激光扫描）、控制器、机器、人员和物联在一起。 |              | 传感器技术、RFID标签、嵌入式系统技术 |
